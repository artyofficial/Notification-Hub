## <img src="https://cdn.discordapp.com/emojis/1234567891.png" width="28" height="28" alt=""/> What is Notifier Hub?

**Notifier Hub** is a **real-time alert system** that monitors popular gambling platforms for **rain events** — random giveaways where free money is dropped in chat.

🔔 The bot detects these events within **milliseconds** and instantly notifies you via Discord, so you can claim your rewards before anyone else.

> **No more refreshing. No more missing out. Just free money, delivered to you.**
